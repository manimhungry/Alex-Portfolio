# Alex Pacheco Santiago Portfolio

Personal mechanical engineering portfolio built with Next.js App Router and Tailwind CSS.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript
- Vercel Analytics and Speed Insights

## Local Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Useful Commands

```bash
npm run lint
npm run build
```

## Project Structure

- `app/page.tsx` - home page
- `app/about/page.tsx` - about page
- `app/projects/page.tsx` - engineering project case studies
- `app/experience/page.tsx` - Leica Biosystems experience page
- `public/` - portfolio media assets
